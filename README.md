Sistema bancario com herança em python

este projeto mostra uma implementação básica de um sistema bancário usando herança em python. existe uma classe principal chamada conta, que guarda dados comuns como número, titular e saldo. a partir dela foram criadas duas contas diferentes: conta corrente, que permite usar limite e ficar com saldo negativo, e conta poupança, que tem rendimento e pode aplicar juros ao saldo.

o banco consegue cadastrar contas, consultar saldo, creditar e debitar valores, fazer transferências e listar as contas existentes. no caso da poupança, também é possível aplicar o rendimento.
a ideia é demonstrar de forma simples como a herança permite reaproveitar código e adicionar comportamentos específicos em cada tipo de conta.
